Conexão com portais

As chaves informadas abaixo ficam apenas na memória desta aba (não são salvas em disco nem enviadas a nenhum servidor além do próprio portal). Enquanto a Petronect não libera sua credencial, o painel opera com dados de demonstração.

Antes de ligar isso à chave real: este protótipo é hospedado como página estática (GitHub Pages). Chamar a API do portal diretamente do navegador expõe a chave a qualquer visitante e normalmente esbarra em bloqueio de CORS do portal. Para produção, essa chamada precisa passar por um pequeno backend/proxy que guarde a chave em segredo. Veja o README do repositório para o passo a passo.
